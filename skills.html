My UX Skills

User Research

In my user reasearch, I try to learn as much as I can about the pains and gains of my users experience. While keeping my goal in mind, I learn new things about my product by listening. This helps me build (or re-build) the best outcome with multiple parties to accommodate for.

User Interface Design

I really enjoy the outlet of my creative drive while designing the user interface. Designing navigation flows and site maps are where I love to "geek out". The best way to design a cohesive product is to work in a team. Collaboration and working as a team are my stregths.

Web Devlopment

I understand the foundational building blocks of HTML, CSS, Javascript and Bootstrap. This understanding is helpful when dicussing design possibilities with programmers and building web-responsive designs.
